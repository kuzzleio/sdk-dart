# Constructor

Use this constructor to create a new instance of the `Http` protocol with specific options.

## Arguments

```dart
HttpProtocol(Uri);
```

<br/>

| Argument  | Type              | Description                  |
| --------- | ----------------- | ---------------------------- |
| `uri`    | <pre>Uri</pre> | URI pointing to a Kuzzle server |


## Return

A `HttpProtocol` protocol instance.
